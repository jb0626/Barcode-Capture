# Barcode-Capture
This project can open website via WebView.

## Prerequisite
'local-web/BarcodeDetector.html' should be run local or wherever on the web server.

## Develop Environment
OS : MAC

IDE : Android Studio

Library : [Quagga2](https://github.com/ericblade/quagga2)

Local web server : http-server

VCS : Git

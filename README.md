# Barcode-Capture
This project can open specific website via WebView.

## Prerequisite
'local-web/BarcodeDetector.html' should be run local or wherever on the web server.

## Develop Environment
OS : MAC

IDE : Android Studio

Local web server : http-server

VCS : Git
